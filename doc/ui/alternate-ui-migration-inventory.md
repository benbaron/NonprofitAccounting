# Alternate UI Migration Inventory

This inventory tracks the current migration state of the alternate JavaFX UI in `org.nonprofitbookkeeping.ui`. It is based on:

- `src/main/java/org/nonprofitbookkeeping/ui/AppPanelId.java`
- `src/main/java/org/nonprofitbookkeeping/ui/PanelHost.java`
- `src/main/java/org/nonprofitbookkeeping/ui/MainWindowAlternate.java`
- currently available legacy panels and actions under `src/main/java/nonprofitbookkeeping/ui/panels` and `src/main/java/nonprofitbookkeeping/ui/actions`

## Status legend

- **Native alternate UI**: an `org.nonprofitbookkeeping.ui` panel or view intended for the alternate shell.
- **Legacy adapter**: an alternate-shell `AppPanel` or direct command that embeds an old `nonprofitbookkeeping.ui` panel.
- **Placeholder**: UI exists but primarily communicates that functionality is not yet implemented or wired.
- **Demo-only**: production UI displays hardcoded realistic business/accounting sample data.
- **Mixed**: combines service-backed behavior with placeholder, demo, direct legacy, or incomplete behavior.
- **Missing**: no first-class alternate panel or workflow exists yet.
- **Dropped/not in scope**: explicitly excluded from the alternate UI migration plan.

## Immediate documentation findings

1. All `AppPanelId` values are represented in `PanelHost.DefaultPanelFactory`, but several are normally bypassed by `WorkspaceRouter` and handled by custom panes in `MainWindowAlternate` (`DASHBOARD`, `SETTINGS`, `BUDGET_EDITOR`, and `SCHEDULES`).
2. The alternate shell still contains production-path hardcoded accounting values in dashboard, ledger, budget, budget-vs-actual, schedules fallback accounts, and assets workflows. These should be addressed by the separate no-demo-data plan step.
3. Company, database, import/export, SCLX import, COA import/export, and database repair are present only as partial/scattered flows or are missing dedicated alternate workflow IDs.
4. Documents & Attachments must remain dropped/not in scope despite the current command-center legacy adapter entry.
