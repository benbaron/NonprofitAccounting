Nonprofit Bookkeeping

The Nonprofit Bookkeeping project provides desktop-oriented accounting workflows for nonprofit organizations, including core bookkeeping, reporting, supplemental data management, and plugin-based extensions.

This page serves as the Javadoc landing page for navigating package documentation across API, services, models, persistence, reporting, UI, and tooling layers.
